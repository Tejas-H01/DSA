# 2469. Convert the Temperature – Java Solution

This repository contains a Java solution for the **LeetCode problem: Convert the Temperature**.

The solution demonstrates how to convert a given temperature in Celsius to **Kelvin and Fahrenheit** using standard conversion formulas.

---

## 📌 Problem Overview

Given a non-negative floating-point number `celsius`, return an array `ans` where:

- `ans[0]` = temperature in **Kelvin**
- `ans[1]` = temperature in **Fahrenheit**

---

## 🧪 Code Functionality

- Converts Celsius to Kelvin using:
  - `Kelvin = Celsius + 273.15`  
- Converts Celsius to Fahrenheit using:
  - `Fahrenheit = Celsius × 1.80 + 32.00`  
- Stores both values in an array  
- Returns the result
