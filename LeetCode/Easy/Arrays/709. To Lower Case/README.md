
# 709. To Lower Case – Java Solution

This repository contains a Java solution for the **LeetCode problem: To Lower Case**.

The solution converts all characters in a string to lowercase using a built-in Java method.

---

## 📌 Problem Overview

Given a string `s`, return the string after replacing every uppercase letter with the same lowercase letter.

---

## 🧪 Code Functionality

- Takes the input string `s`  
- Uses the built-in `toLowerCase()` method  
- Converts all uppercase characters to lowercase  
- Returns the updated string
