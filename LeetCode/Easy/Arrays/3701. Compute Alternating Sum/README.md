# 1911. Maximum Alternating Subsequence Sum – Java Solution

This repository contains a Java solution for the **LeetCode problem: Maximum Alternating Subsequence Sum**.

The solution computes an alternating sum by adding elements at even indices and subtracting elements at odd indices.

---

## 📌 Problem Overview

Given an integer array `nums`, return the alternating sum of its elements.

The alternating sum is defined as:
- Add elements at even indices  
- Subtract elements at odd indices  

---

## 🧪 Code Functionality

- Initializes a variable `sum` to store the result  
- Traverses the array using a loop  
- If the index is even, adds the element to `sum`  
- If the index is odd, subtracts the element from `sum`  
- Returns the final computed alternating sum
