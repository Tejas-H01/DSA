# 1351. Count Negative Numbers in a Sorted Matrix – Java Solution

This repository contains a Java solution for the **LeetCode problem: Count Negative Numbers in a Sorted Matrix**.

The solution counts the total number of negative elements using a straightforward traversal approach.

---

## 📌 Problem Overview

Given a `m x n` matrix `grid` which is sorted in non-increasing order both row-wise and column-wise,  
return the total number of negative numbers in the matrix.

---

## 🧪 Code Functionality

- Initializes a counter `neg` to store the number of negative elements  
- Uses two nested loops to traverse the matrix row by row  
- Checks each element to see if it is less than zero  
- Increments the counter whenever a negative number is found  
- Returns the final count
