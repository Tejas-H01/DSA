# 2544. Alternating Digit Sum – Java Solution

This repository contains a Java solution for the **LeetCode problem: Alternating Digit Sum**.

The solution converts the number into a string and computes the alternating sum of its digits based on their positions.

---

## 📌 Problem Overview

Given a positive integer `n`, return the alternating sum of its digits.

The alternating sum is defined as:
- Add digits at even indices  
- Subtract digits at odd indices  

---

## 🧪 Code Functionality

- Converts the integer `n` into a string  
- Initializes a variable `sum` to store the result  
- Iterates through each character (digit) in the string  
- Converts each character back to an integer  
- Adds the digit if the index is even  
- Subtracts the digit if the index is odd  
- Returns the final alternating sum
